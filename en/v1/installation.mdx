## 1. Install Rust

### macOS / Linux
Run this command in your terminal:
<CodeBlock language="bash">{`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`}</CodeBlock>

### Windows
Download and run the <a href="https://win.rustup.rs" target="_blank" rel="noopener">Rust installer for Windows</a>.

## 2. Follow the Prompts
- Proceed with the <strong>default installation</strong> (recommended for most users).
- Press <strong>Enter</strong> to confirm.
- The installer will handle the rest.

## 3. Restart Your Terminal
To ensure your shell recognizes the new commands, <strong>close and reopen</strong> your terminal.

## 4. Verify Your Installation
<CodeBlock language="bash">{`rustc --version`}</CodeBlock>
<CodeBlock language="bash">{`cargo --version`}</CodeBlock>
If you see something like <code>rustc 1.77.0</code>, youâ€™re golden! ğŸŒŸ

## 5. (Optional) Add Rust to PATH
If you get a <code>command not found</code> error after installation:

### macOS / Linux
<CodeBlock language="bash">{`source $HOME/.cargo/env`}</CodeBlock>

### Windows
Restart the terminal or add <code>%USERPROFILE%\.cargo\bin</code> to your system environment variables.

## 6. Keeping Rust Up to Date
To update Rust:
<CodeBlock language="bash">{`rustup update`}</CodeBlock>

To install nightly or switch versions:
<CodeBlock language="bash">{`rustup install nightly`}</CodeBlock>
<CodeBlock language="bash">{`rustup default stable`}</CodeBlock>
